## 職務要約

React、TypeScript、Next.js を軸としたモダンなフロントエンド技術を中心に、Go によるバックエンド開発も含めたフルスタック開発経験を有しています。直近 2 年間で商用レベルの Web アプリケーション開発に携わり、Next.js の App Router やアトミックデザインによるコンポーネント設計など、保守性とパフォーマンスを重視した実装に取り組んできました。

フロントエンド開発では TypeScript による型安全性の確保、再利用可能なコンポーネント設計、カスタムフックによるロジック分離など、品質向上のための実践を重ねています。バックエンド開発では REST API 設計・開発、認証システム対応、外部サービス連携の経験があり、要件定義から設計、開発、テストまで一貫して担当できる実績があります。

前職では救急救命士として緊急時の判断力や問題解決能力を培い、それらを IT 分野でのトラブルシューティングや優先順位付けに活かしています。自走力を持ってキャッチアップし、ユーザー視点を大切にした価値提供に注力しています。
